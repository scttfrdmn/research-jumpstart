# Crystal Structure Analysis - Tier 3 Starter

Basic crystallography and materials property analysis.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate crystal-structure

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Crystal systems and lattice parameters
- Structure visualization
- Density and property calculations
- Band gap analysis
- Material classification

## Sample Data

10 common materials with crystal structure data.
