# Disease Surveillance Basics - Tier 3 Starter

Introduction to epidemiology through infectious disease monitoring.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate disease-surveillance

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Incidence and prevalence calculation
- Epidemic curve visualization
- Outbreak detection methods
- Basic reproduction number (R0)
- Geographic disease mapping

## Sample Data

500 case reports over 12 months:
- 5 regions with varying populations
- Age, symptom onset, demographics
- Realistic outbreak patterns
