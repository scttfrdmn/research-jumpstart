# Medical Image Fundamentals - Tier 3 Starter

Introduction to medical image analysis using chest X-ray data.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate medical-imaging

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Medical image feature extraction
- Image quality metrics
- Feature-based classification
- ROC curve analysis
- Diagnostic performance evaluation

## Sample Data

100 chest X-ray feature vectors:
- 60 Normal cases
- 40 Pneumonia cases
- 9 extracted features per image (intensity, texture, anatomy)
