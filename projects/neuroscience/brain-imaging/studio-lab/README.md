# Brain Imaging Analysis - Tier 3 Starter

fMRI data analysis and brain connectivity.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate brain-imaging

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- fMRI time series analysis
- Brain region connectivity
- Statistical mapping
- Activation patterns
- Network analysis
