# Artifact Measurement - Tier 3 Starter

Introduction to archaeological data analysis through artifact morphometrics.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate artifact-analysis

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Artifact measurement and documentation
- Morphometric analysis
- Type classification
- Temporal trends
- Spatial distribution patterns

## Sample Data

150 artifact records from excavation:
- Pottery, lithics, and metal objects
- Length, width, weight measurements
- Provenance and dating information
