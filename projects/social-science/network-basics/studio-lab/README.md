# Social Network Basics - Tier 3 Starter

Introduction to social network analysis using a small friendship network.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate social-network

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Network visualization
- Centrality measures (degree, betweenness, closeness, eigenvector)
- Community detection
- Path analysis
- Network metrics

## Sample Data

Small friendship network:
- 10 people (nodes)
- 38 friendships (edges)
- Weighted relationships
