# Language Statistics - Tier 3 Starter

Introduction to computational linguistics through statistical language analysis.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate language-stats

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Word frequency distributions
- Zipf's law and power laws
- Type-token ratio (lexical diversity)
- N-gram analysis
- Language comparison

## Sample Data

Text corpora from 4 languages:
- English, Spanish, German, Japanese
- ~1000 words per language
- Parallel content for comparison
