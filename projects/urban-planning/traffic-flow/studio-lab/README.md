# Traffic Flow Analysis - Tier 3 Starter

Introduction to urban analytics through traffic pattern analysis.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate traffic-flow

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Traffic volume and speed analysis
- Congestion metrics (Level of Service)
- Time-of-day patterns
- Weekday vs weekend differences
- Capacity utilization

## Sample Data

2000+ hourly observations from 6 intersections:
- Vehicle counts and speeds
- 4 weeks of continuous monitoring
- Rush hour and off-peak periods
