# Weather Pattern Analysis - Tier 3 Starter

Climate science fundamentals through weather station data analysis.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate weather-analysis

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Time series decomposition
- Seasonal pattern analysis
- Trend detection
- Anomaly identification
- Climate normals calculation
