## What You'll Learn

- Transit photometry analysis (light curves)
- Radial velocity method for planet detection
- Period-finding algorithms (Lomb-Scargle)
- Planet parameter estimation (radius, mass, orbit)
- Stellar characterization
- Phase-folding and transit modeling

## Key Analyses

1. **Light Curve Processing**: Detrending, normalization
2. **Transit Detection**: Box-Least-Squares (BLS) algorithm
3. **Planet Parameters**: Radius, period, semi-major axis
4. **Radial Velocity**: Orbital mechanics, mass determination
5. **False Positive Elimination**: Statistical validation
