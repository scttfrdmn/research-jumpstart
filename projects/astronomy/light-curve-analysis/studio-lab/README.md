# Light Curve Analysis - Tier 3 Starter

Introduction to observational astronomy through variable star analysis.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate light-curve

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Light curve visualization
- Period detection (Lomb-Scargle)
- Phase folding
- Variability classification
- Time series analysis

## Sample Data

5 variable stars with ~200 observations each:
- Cepheid variables
- RR Lyrae stars
- Eclipsing binaries
- Realistic noise and sampling
