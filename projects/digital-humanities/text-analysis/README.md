## What You'll Learn

- Load and process historical texts
- Frequency analysis and word distributions
- Topic modeling for thematic analysis
- Stylometric analysis (author attribution)
- Named entity recognition (people, places, events)
- Temporal language change tracking
- Text similarity and comparison

## Key Analyses

1. **Lexical Analysis**
   - Word frequency distributions
   - Vocabulary richness (TTR, Herd's D)
   - Hapax legomena identification

2. **Topic Modeling**
   - LDA for theme extraction
   - Topic evolution over time
   - Thematic comparison across texts

3. **Stylometry**
   - Function word analysis
   - Sentence length patterns
   - Authorship attribution

4. **Named Entity Recognition**
   - Extract people, places, organizations
   - Build knowledge graphs
   - Track entity mentions over time
