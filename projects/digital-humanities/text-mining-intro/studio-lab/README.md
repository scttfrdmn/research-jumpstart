# Text Mining Introduction - Tier 3 Starter

Introduction to digital humanities through historical text analysis.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate text-mining

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Text preprocessing and tokenization
- Word frequency analysis
- TF-IDF scoring
- Topic identification
- Temporal analysis

## Sample Data

20 historical texts (1850-1900):
- Industrial revolution period documents
- Authors, years, and full text
- Realistic historical vocabulary
