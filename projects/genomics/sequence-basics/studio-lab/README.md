# DNA Sequence Analysis Basics - Tier 3 Starter

Introduction to genomics through DNA sequence analysis and phylogenetics.

## Quick Start

```bash
# Create environment
conda env create -f environment.yml
conda activate genomics-basics

# Launch Jupyter
jupyter lab quickstart.ipynb
```

## Topics Covered

- Sequence alignment (multiple sequence alignment)
- Phylogenetic tree construction
- Genetic diversity metrics
- Variant identification
- Population structure

## Sample Data

10 DNA sequences (COI gene, 240 bp):
- 4 geographic locations
- Realistic polymorphisms
- FASTA format
